## Decisions Made During Build

- Stack is the PRD's (Postgres/Drizzle/Hono), not Convex, despite Convex-tooled environment.
- Single-user → token auth (Better Auth removed); content not linked to a user.
- There is one workspace team. Team members are content entities assigned to projects via `project.teamMemberIds`, not access control.
- Reads are client-side via `useHonoQuery` (not Server Components); detail derives from the list query.
- Realtime via Redis → Bun WS gateway; mutations publish, never invalidate directly.
- Ports are 700x (app 7000, gateway 7001) — 3000 was taken on this machine.
- Dark mode is class-based (next-themes); semantic tokens make both themes automatic.
- Copy feedback is a **sonner** toast (theme-aware `Toaster` in root layout), not an inline check-mark swap. All copy sites go through `copyText()` (`src/lib/clipboard.ts`); the old `useCopy` hook was removed.
